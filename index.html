CSS Selectors and Specificity

CSS selectors are used to select the HTML elements you want to style.

If several selectors applied to the same element,
the browser use the Specificity of the selector to decide which CSS styling to display.
A best practice is to style elements using the lowest degree of specifity,
so if an element needs a new style, it is easy to override.

Specificity Basic Rules:

  • Specificity Hierarchy:
    1. inline style (the highest specificity level)
    2. IDs
    3. Classes
    4. Elements (the lowest specificity level)
  • If the selectors are the same then the last one will be applied.
  • !important will override any selector (don't overuse it!).
